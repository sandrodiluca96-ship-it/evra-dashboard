# EVRA Dashboard

Versione completa con logo EVRA.

## File inclusi
- `app.py`
- `requirements.txt`
- `data/commesse.xlsx`
- `data/reparti.xlsx`
- `assets/evra_logo.svg`

## Avvio locale

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Aggiornamento su Streamlit Cloud

Caricare/sostituire nel repository GitHub:
- `app.py`
- `requirements.txt`
- cartella `data`
- cartella `assets`

La dashboard si chiama solo: **EVRA Dashboard**.
